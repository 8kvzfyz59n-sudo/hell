教學

上半身原型:製圖邏輯
  1. 畫基礎長方形:寬=身幅(B/2+6)、高=背長;從上端(A點)往下 B/12+13.7 畫出胸圍線 BL。
  2. BL 上取背幅(B/8+7.4)與胸幅(B/8+6.2),中間即袖窿範圍;F 與 C 的中點垂直向下為脇線。
  3. 領口:前領寬◎=B/24+3.4、前領深◎+0.5、後領寬◎+0.2;肩斜前22°、後18°。
  4. 胸省角度=(B/4−2.5)°,從 BP 往袖窿方向打開;後肩省=B/32−0.8,指向 E 點。
  5. 總腰省=身幅−(W/2+3),按 a14%/b15%/c11%/d35%/e18%/f7% 分配到六個位置。
  6. 製作前先以胚布試穿,依體型補正後再作為打版母型。
袖原型:製圖邏輯
  1. 袖山高=(前後肩點平均高度到 BL 距離)×5/6,袖山越高袖越合身、活動量越小。
  2. 前袖斜線長=前AH、後袖斜線長=後AH+1(B≥85 再加 0.1~0.3),由此定出袖幅。
  3. 袖山弧線縮縫份(いせ)約為 AH 的 5~8%,縫製時用兩道疏縫抽縮、以熨斗歸攏。
  4. 合印:前袖對前身、後袖對後身,袖山點對肩線。

教學

直筒裙:製圖邏輯與工法(隱形拉鍊+貼邊/腰頭)
  1. 基礎長方形:寬=H/2+2(臀鬆份整圈4cm)、長=裙長;WL 下腰長處畫 HL。
  2. 脇線取 HL 中點往後移 1cm(前片較寬,符合體型)。
  3. 腰目標:後=W/4、前=W/4+1(含鬆份與前後差);與臀寬的差=縮減量D。
  4. D 的一半在脇邊收(脇腰點上抬1cm畫順),另一半做2個腰省(WL三等分處,後省13/12cm、前省9/10cm)。
  5. 隱形拉鍊:開在後中心,開口=腰長+1cm(過 HL 才穿得進去);先縫拉鍊再縫剩餘後中心線,齒務必燙開再車。
  6. 腰部收邊二選一:
    ①腰頭:版片 W+3 × 3cm(對摺+燙襯),3cm 是疊份鈕釦位。
    ②內貼邊:複製版型腰線下7cm、把腰省合併後的形狀,燙襯後與腰線正面相對車縫翻入。
  7. 後開衩=裙長/3(裙長60→衩20),走路才有活動量;衩持出約4cm。
  8. 縫份建議:腰1、脇/後中心1.5、下襬4(直筒裙下襬用千鳥縫或奇縫固定)。

圓裙(版片=半件,對摺裁或裁2片)

計算結果

教學

圓裙:製圖邏輯
  1. 原理:讓腰線弧長=腰圍。n/4 圓裙的腰半徑 r=2W/(nπ);全圓 r≈W/6.28、半圓 r≈W/3.14。
  2. 下襬半徑=r+裙長,兩弧之間就是裙片;本站畫「半件」(圓心角=n×45°),對摺裁或裁2片。
  3. 圓越大裙襬越澎、用布越多;全圓裙下襬總長=2π(r+裙長)。
  4. 斜布方向會鬆垂:裁好先吊掛一天,再把下襬修齊。
  5. 腰部處理與直筒裙相同(腰頭或貼邊+隱形拉鍊);半圓以下也可改鬆緊帶腰。

直筒褲原型(前片+後片)

計算結果

教學

直筒褲:製圖邏輯
  1. 前片基礎長方形:寬=H/4+2(鬆份)、縱向取股上與腰長(HL);前襠寬=前片寬/4−1.5。
  2. 摺山線(燙線)=脇到襠尖的中點垂直線;股下=褲長−股上;膝線 KL=股上+股下/2−7。
  3. 褲口與 KL 以摺山線左右對稱,前褲口半寬=摺山−3.5;後片 KL 與褲口每側比前片加1.5。
  4. 後襠寬=前襠寬+(前片寬/4−1);後中心腰點上抬2.5、內收2(配合坐姿活動量)。
  5. 腰:前=W/4+1、後=W/4;脇收2.5並上抬1.2;剩餘做腰省(超過3cm自動分兩支,前9/7.5、後12/10cm)。
  6. 前襠彎經對角導引點2/3處畫順、後襠彎更深;縫合後前後襠必須成一條連續順滑曲線,拿曲線尺對圖確認。
  7. 長度選擇:長褲=褲長設定值;5分褲=股上+股下/2;3分熱褲=股上+10cm。短褲版本襠彎與臀圍完全相同,只是提前落褲口。
  8. 拉鍊版收腰:左脇隱形拉鍊(開口=腰長+2),腰頭或貼邊做法同直筒裙分頁。
鬆緊帶版:差異與做法
  1. 版型差異:無腰省、無脇收、後中心不上抬——腰口=臀圍寬直上,才能套過臀部穿脫,免拉鍊。
  2. 腰口處理:上緣外加7cm縫份(3.5cm×2)向內折兩次車縫,留3cm返口穿鬆緊帶。
  3. 鬆緊帶長≈腰圍×0.9(織帶彈性不同,實穿微調);穿入後頭尾重疊2cm車Z字固定,再車合返口。
  4. 建議兩道車線把鬆緊帶分上下兩格,不易翻捲。
  5. 適合針織或有垂墜感的梭織布;硬挺布做鬆緊腰會蓬起,打樣時注意。

A4 試摺紙型(印出來先摺摺看)

紙型上的線條:紅色一點一劃的線=山摺(凸起來,摺痕朝自己);藍色虛線=谷摺(凹下去,往內摺)。英文對照見下方教學。

計算結果

教學

這個計算機在算什麼?(白話版)
  1. 百摺裙就是把一塊「比腰圍寬很多」的長方形布,一摺一摺收到剛好圍住腰。這裡幫你算:那塊布到底要多寬。
  2. 兩個關鍵名詞:陽摺=摺好後表面看得到的那一段寬度;陰摺=摺進去、藏在下面看不到的深度。
  3. 做一個摺要吃掉的布,依摺法不同:刀褶(摺痕都往同一邊)=陽摺+2×陰摺;箱褶(兩摺相對,像禮盒包裝的摺法)=陽摺+4×陰摺;手風琴褶(一凸一凹交錯)=陽摺+陰摺。
  4. 要摺幾個?用(腰圍+鬆份)除以「一個摺在腰上佔的寬度」,小數無條件進位——寧可多一點,不能圍不住。
  5. 布的總寬=摺數×每摺吃掉的布,再加後面開口的縫份。這就是「整塊裙子布料需要多寬」的答案。
  6. 上下方向:裙長+腰頭+摺邊縫份,如果小於買來的布幅寬,布橫著擺就能一整片裁完,不用接布。
  7. 陽摺越窄、陰摺越深=皺摺越密越澎;陽摺越寬、陰摺越淺=越平順。拿不定主意就先印試摺紙型摺摺看。
A4 試摺紙型:怎麼印、怎麼摺(含英文對照)
  1. 按「下載 PDF」印出來。列印時一定要選「實際大小」或「100%」,不要選「符合頁面大小」,不然尺寸會跑掉。
  2. 印好先量最下面那條尺:上面寫 PRINT CHECK 的線段應該剛好 10 公分。不是 10 公分就是印表機縮放了,回去改設定重印。
  3. 紙上英文對照:MOUNTAIN(紅色一點一劃線)=山摺,摺起來凸向自己、摺痕在表面;VALLEY(藍色虛線)=谷摺,往內凹摺;CUT ALONG FRAME=沿黑框剪下;FOLDED WIDTH=全部摺完應該剛好這麼寬(拿摺好的紙對上去驗證);數字 1、2、3 是摺的順序;小箭頭是摺份要壓過去的方向。
  4. 箱褶印出來是「工字褶」(兩條摺線在表面相接);想要外凸盒子那種一般箱褶,把紙翻面照摺就是了。
  5. 摺完覺得太密、太淺、不夠澎——回來改「陽摺寬」「陰摺深」的數字,重印再摺,滿意了再去裁布。紙很便宜,布不便宜。

男裝上衣原型:製圖邏輯
  1. 男子原型採「胸度式」:大部分尺寸由胸圍 C 與背長兩個數字決定;後中心長=背長+0.5。
  2. 基礎長方形:寬=衣寬(C/2+6.7);從上端 A 往下 C/6+8.5 畫胸圍線 BL。
  3. BL 上取背幅(C/6+5.8)與胸幅(C/6+2.9),兩幅線中點垂直向下為脇線。
  4. 領口:前領寬=C/16+1.9、前領深=領寬+0.5、後領寬=前領寬+0.3;肩斜前 22°、後 21°。
  5. 男子原型無胸省;後肩褶=C/32,指向背幅中段的 I 點。
  6. 總腰省=(C/2+6.7)−(W/2+4),按 a16%/b16%/c36%/d24%/e8% 分配(e 為後中心縮份)。
  7. 製作前先以胚布試穿,依體型補正後再作為打版母型。
男裝袖原型:製圖邏輯
  1. 袖山高=(前後肩點平均高度到 BL 距離)×5/6。
  2. 前袖斜線=前AH、後袖斜線=後AH+1,由此定出袖幅;肘線=袖長/2+2.5。
  3. 袖山縮縫份(いせ)縫製時用兩道疏縫抽縮、以熨斗歸攏。

教學

男裝西裝褲(單褶):怎麼畫出來的(白話版)
  1. 輸入都填實際量到的尺寸就好:公式裡已經幫你加了活動需要的鬆量(例如臀圍那條 H/4+2.5,那個 2.5 就是鬆量)。
  2. 長度:股上長=腰到胯下(含 3cm 腰頭,身片實際畫股上長−3);股下長=胯下到褲腳;褲長=兩者相加。膝蓋位置(中檔線)取股下的一半再往上 5cm。
  3. 整套圖用一個基準值 ☆=臀圍÷12:臀圍線在橫檔線(胯下那條水平線)上方 ☆;前片褲襠尖端往外伸 ☆×2/3;後片再多伸 ☆/2,並往下落 1cm(坐下才有空間)。
  4. 前片有一個「褶」壓在挺縫線(褲管燙出來那條直線)上:褶寬不是固定值,是腰口剩多少收多少(前片寬−前中心收 0.7−側縫收 1.5−腰圍÷4)。
  5. 後片腰口多出來的量改用兩個「省」收掉(一大一小差 0.6),省尖指向後口袋;後中心腰點往上翹 1cm(后翘),彎腰坐下才不會扯。
  6. 褲腳:前片=前片寬−3.5、後片比前片寬 1;膝蓋處各再寬 1,自然直筒(H 型)。
  7. 腰頭另外裁:寬 3cm,左片=腰圍/2+3、右片=腰圍/2+6(多的 3cm 是扣子搭份),圖下方兩條就是。
  8. 紅線是記號不是裁切線:PLEAT=褶、DART=省、POCKET=口袋位置、FLY=前門襟(拉鍊開口)、CREASE=挺縫線。
