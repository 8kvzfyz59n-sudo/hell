# 新文化式原型產生器

依日本文化服裝學院「新文化式(2000年版)成人女子原型」製圖法,輸入 **胸圍 B、腰圍 W、背長**(袖原型另需袖長),自動產生上半身原型與袖原型,可下載 SVG 與 1:1 實寸向量 PDF。

## 使用方式

直接用瀏覽器開啟 `index.html` 即可,完全離線可用(PDF 由內建向量產生器輸出,不依賴外部函式庫)。
要公開部署,把整個資料夾上傳到任何靜態空間(GitHub Pages、Cloudflare Pages、Netlify)即可。

## 檔案結構

```
index.html      介面
css/style.css   樣式
js/app.js       製圖計算 + SVG 繪製 + 下載(可獨立以 Node 測試)
```

## 採用公式(新文化式)

| 部位 | 公式 |
|---|---|
| 身幅 | B/2 + 6 |
| A~BL(袖窿深) | B/12 + 13.7 |
| 背幅 | B/8 + 7.4 |
| 胸幅 | B/8 + 6.2 |
| BL~前身片上端 | B/5 + 8.3 |
| 前領口幅 ◎ | B/24 + 3.4 |
| 前領口深 | ◎ + 0.5 |
| 後領口幅 | ◎ + 0.2 |
| 前/後肩斜 | 22° / 18° |
| 後肩省 | B/32 − 0.8 |
| 胸省角度 | (B/4 − 2.5)° |
| 總腰省量 | 身幅 − (W/2 + 3) |
| 腰省分配 a~f | 14% / 15% / 11% / 35% / 18% / 7% |
| 袖山高 | (前後SP平均高~BL) × 5/6 |
| 前袖斜線 | 前AH |
| 後袖斜線 | 後AH + 1(B≥85 另加 0.1~0.3) |

計算值已與文化購買事業部公開之各部寸法速查表(B82~88)核對一致。

## 已知限制

- 領口、袖窿、袖山弧線以教材導引點 + 平滑曲線近似,與手繪 D 彎尺結果會有毫米級差異。
- B ≥ 90 時胸省閉合後前袖窿會出角,教材要求手動修順(頁面會提示)。
- 新文化式袖山縮縫份(いせ)本來就偏多(約 6~8%),縫製時需以兩道疏縫抽縮處理。
- 原型僅為打版基礎,實際商品開發仍需胚衣試穿補正。
